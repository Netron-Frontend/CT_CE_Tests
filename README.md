# Подготовка к ЦТ и ЦЭ 2025

Сайт для подготовки к централизованному тестированию (ЦТ) и централизованному экзамену (ЦЭ) с тестами 2025 года.

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **TailwindCSS** - стилизация
- **App Router** - маршрутизация Next.js

## Установка

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev сервер:

```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Структура проекта

```
├── app/                    # Next.js App Router
│   ├── page.tsx           # Главная страница
│   ├── test/[id]/         # Страница теста
│   ├── layout.tsx         # Корневой layout
│   └── globals.css        # Глобальные стили
├── components/            # React компоненты
│   ├── TestQuestion.tsx   # Компонент вопроса
│   └── TestResults.tsx    # Компонент результатов
├── data/                  # Данные тестов
│   └── tests.ts           # Массив тестов
├── types/                 # TypeScript типы
│   └── test.ts            # Типы для тестов
└── public/                # Статические файлы
```

## Доступные тесты

### ЦТ (Централизованное тестирование)

- Математика
- Русский язык
- Физика

### ЦЭ (Централизованный экзамен)

- Беларуская мова
- История Беларуси

## Функциональность

- ✅ Выбор теста по типу (ЦТ/ЦЭ) и предмету
- ✅ Прохождение теста с навигацией между вопросами
- ✅ Отслеживание времени прохождения
- ✅ Визуальная индикация отвеченных вопросов
- ✅ Детальные результаты с объяснениями
- ✅ Адаптивный дизайн
- ✅ Поддержка темной темы

## Добавление новых тестов

Для добавления новых тестов отредактируйте файл `data/tests.ts` и добавьте новый объект в массив `tests`:

```typescript
{
  id: "unique-id",
  type: "CT" | "CE",
  subject: "Название предмета",
  year: 2025,
  duration: 180, // в минутах
  questions: [
    {
      id: 1,
      text: "Текст вопроса",
      options: ["Вариант 1", "Вариант 2", "Вариант 3", "Вариант 4"],
      correctAnswer: 0, // индекс правильного ответа (0-based)
      explanation: "Объяснение ответа"
    }
  ]
}
```
